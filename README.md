# ULTRAC00L
## Présentation
Projet de deuxième année de licence informatique, a l’Université du Mans.
Codé en langage C, Jeu 2d, inspiré de Fps 3d, notamment ULTRAKILL.
### Outils utilisés :
- Utilisation de SDL3 pour l'interface graphique
- Utilisation de nob.h pour la compilation des fichier .c
- Utilisation de Doxygen pour la documentation du code
